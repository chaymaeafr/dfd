About this project
This project is a site that I have created while studying html canvas. I started by watching Interactive developer Jongmin Kim's YouTube lecture. He is my favorite and respected interactive designer. I started canvas studying after seeing his works. I wanted to create a site where I could see things made while studying canvas. I plan to add additional work created while studying canvas to this site. If you have any other questions, please contact me by email.
Contact: jjangikk@gmail.com
